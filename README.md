# Editor
A simple texteditor written in c.

## Controls
- CTRL+C => close
- CTRL+S => save

to open a file run ```editor <filename>```

## Dependencies
- ```ncurses``` libraray, install it by running:
```sh
apt install libncurses5-dev
```

## Compile & Run
```sh
make
./editor
```
or
```sh
./editor <filename>
```
to open a file.
